# CarDiagnostic
# Проект был выполнен в качестве задания на сессии
Этот проект представляет собой десктопное приложение для диагностики автомобилей на основе экспертной системы, разработанное на Python. Оно позволяет пользователю выбирать симптомы неисправности автомобиля, после чего система на основе базы знаний (SQLite) определяет наиболее вероятные неисправности, выдаёт подробную информацию о них, а также рекомендации по ремонту. Интерфейс реализован с использованием библиотеки Tkinter, а логика диагностики – на основе правил «симптом → неисправность» с учётом уверенности и комбинационных правил.
